JPA : Java Persistence Api : Interface de mapping modèle de données relationnel / objets java

Hybernate : implementation Java de JPA. Hybernate repose sur un driver JDBC pour communiquer avec la base de données MySQL.
Nous utilisons ici la version 5.6.0 du package hibernate-core.

Driver JDBC:  pilote java d'interface base de données. Chaque base de données à on propre pilote.
nous utilisons le pilote JDBC MySQL  "mysql-connector-java" version 8.0.25 ( pour Java 8)
